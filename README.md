# cs4261
## iOS app
To run iOS app, install cocoapod with


`sudo gem install cocoapods`


Then go to the project directory in terminal with 


`cd cs4261/CS4261Assignment1`


And run `pod install`


Open "CS4261Assignment1.xcworkspace" and hit run
